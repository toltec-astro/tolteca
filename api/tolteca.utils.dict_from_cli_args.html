dict_from_cli_args¶

tolteca.utils.dict_from_cli_args(args)[source]¶

Return a nested dict composed from CLI arguments.

This is used to compose config dict on-the-fly. Nested keys can be specified using syntax like --a.b.c. Nested lists are supported with the index as the key: --a.0.c. The values of the options are parsed as YAML string.
